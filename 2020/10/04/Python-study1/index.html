python注释

  • 用#

    python没有多行注释的语法，要是想要多行注释，就每行加入一个#

  • 添加多行字符串

    python将忽略未分配给变量的字符串文字

字符串没有分配给变量，python会读取代码，然后忽略

python变量

python中的变量是在赋值的时候创造的，不需要单独定义

python中没有声明变量的命令

也可以在设置后更改类型

字符串变量可以使用单引号或双引号声明

python变量命名规则

  • 变量名必须以字母或下划线字符开头
  • 变量名称不能以数字开头
  • 变量名只能包含字母数字字符和下划线（A-z、0-9 和 _）
  • 变量名称区分大小写（age、Age 和 AGE 是三个不同的变量）

变量名称区分大小写

变量赋值

  • 能再一行中为多个变量赋值

  • 能在一行中为多个变量分配相同的值

输出语句

直接使用print，如需结合文本和变量或变量与变量，Python 使用 + 字符

全局变量与局部变量

使用global关键字，可将局部变量变为全局变量

也可以用作在函数内部改变全局变量的值

python数据类型

可以使用type()函数获取对象的数据类型

文本类型： str
数值类型： int, float, complex
序列类型： list, tuple, range
映射类型： dict
集合类型： set, frozenset
布尔类型： bool
二进制类型： bytes, bytearray, memoryview
示例 数据类型
x = “Hello World” str
x = 29 int
x = 29.5 float
x = 1j complex
x = [“apple”, “banana”, “cherry”] list
x = (“apple”, “banana”, “cherry”) tuple
x = range(6) range
x = {“name” : “Bill”, “age” : 63} dict
x = {“apple”, “banana”, “cherry”} set
x = frozenset({“apple”, “banana”, “cherry”}) frozenset
x = True bool
x = b”Hello” bytes
x = bytearray(5) bytearray
x = memoryview(bytes(5)) memoryview

设置特定数据类型，可以自己使用构造函数

示例 数据类型
x = str(“Hello World”) str
x = int(29) int
x = float(29.5) float
x = complex(1j) complex
x = list((“apple”, “banana”, “cherry”)) list
x = tuple((“apple”, “banana”, “cherry”)) tuple
x = range(6) range
x = dict(name=”Bill”, age=36) dict
x = set((“apple”, “banana”, “cherry”)) set
x = frozenset((“apple”, “banana”, “cherry”)) frozenset
x = bool(5) bool
x = bytes(5) bytes
x = bytearray(5) bytearray
x = memoryview(bytes(5)) memoryview

将变量指定类型-Python Casting

  • int() - 用整数字面量、浮点字面量构造整数（通过对数进行下舍入），或者用表示完整数字的字符串字面量
  • float() - 用整数字面量、浮点字面量，或字符串字面量构造浮点数（提供表示浮点数或整数的字符串）
